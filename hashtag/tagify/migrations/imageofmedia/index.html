/allmembers/members post get

Add new record for user signup

post

Create signup

get

Fetch user detail

post /allmembers/members

Create signup

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • first_name: required (string)

    First name

  • last_name: required (string)

    Last name

  • email: required (string - pattern: ^.+@.+\..+$)

    Email id (unique)

  • mobile: required (string)

    Enter mobile no len 10 digit and it should be NUMERIC(not alpha numeric)

  • max_age: (integer - maximum: 100)

    Common age field, extract max age like age=10-20 then max_age=20 , min_age=10

  • min_age: (integer - minimum: 0)

    Common age field, extract max age like age=10-20 then max_age=20 , min_age=10

  • gender: required (string)

    Select gender({key:value}) {male:male, female:female, other:other}

  • role: required (integer)

    Select role

  • membership: required (object)
    • subscribed_date: (date-only)

      Select date (required if payment method is selected)

    • initial_amount: (integer)

      Enter amount

    • offer: (integer - maximum: 11)

      Enter offer

    • payment_method: (string)

      Select payment method ({key:value}) {Cash:cash, Website:website, Link:link}

    • pioneer_duration: (string)

      Select duration ({key:value}) {Monthly:monthly, Yearly:yearly}

    • membership_code: (string)

      Enter membership code

  • userOtherDetail: required (object)
    • hotspot: (boolean)

      Tick on checkbox

    • whatsapp: (boolean)

      Tick on checkbox

    • insurance: (boolean)

      Tick on checkbox

    • note: (string)

      Textarea for note

    • looking_for: (array)

      Multi select ({key:value}) {Co-Founder:co-founder, Customer:customer, Investor:investor}

    • open_co_found: (boolean)

      Tick on checkbox

{
  "first_name": "varun",
